# Travel Management System

## Overview

Travel Management System is a web-based application developed using Java, JSP, Servlets, JDBC, MySQL, and Apache Tomcat. The system allows users to register, log in, browse travel packages, view hotels, and make travel bookings.

## Features

* User Registration
* User Login Authentication
* Travel Package Management
* Hotel Management
* Package Booking System
* MySQL Database Integration
* Session Management

## Technologies Used

### Frontend

* JSP
* HTML
* CSS

### Backend

* Java Servlets
* JDBC

### Database

* MySQL

### Server

* Apache Tomcat 9

## Project Structure

* Model Layer
* DAO Layer
* Servlet Layer
* JSP Pages
* MySQL Database

## Database Name

travel_management

## Setup Instructions

1. Clone the repository.
2. Import into Eclipse as a Dynamic Web Project.
3. Configure Apache Tomcat 9.
4. Add MySQL Connector JAR.
5. Execute the SQL script to create the database.
6. Run the project on Tomcat.
